## 9. WORKFORCE PLANNING MODEL

### 9.1 Planning Tools Currently Used

1. **DHG (Dotation Horaire Globale)** - Secondary teacher planning
2. **Effectifs Prévisionnels** - AEFE long-term projection model

### 9.2 Planning Formula - Secondary

```
Total Hours = Classes × Hours per Subject per Week
Teacher FTE = Total Hours ÷ 18 (standard hours)
```

### 9.3 Planning Formula - Primary

```
Teachers = Number of Classes × 1.0
+ Language/Specialist Teachers (30-40% additional)
```

### 9.4 Gap Analysis (TRMD)

The TRMD compares:
- **Besoins (Needs):** Hours required from DHG
- **Apport Initial:** Available teacher hours
- **Deficit:** Gap to fill with recruitment or overtime
- **HSA:** Overtime allocation
