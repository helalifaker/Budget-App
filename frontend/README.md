# EFIR Budget Planning Frontend

React 19 + Vite 7 + TypeScript 5.9 scaffold aligned to the v1.2 TSD.

## Scripts
- `npm run dev` — start dev server (default port 5173).
- `npm run build` — production build.
- `npm run preview` — preview built app.
- `npm run lint` — ESLint 9 flat config.
- `npm run format` — Prettier 3 check.
- `npm run typecheck` — `tsc --noEmit`.
- `npm run test` — Vitest + Testing Library (jsdom).

## Notes
- React Query is prewired for data fetching/caching.
- Tailwind/Shadcn can be added next by installing Tailwind 4 plugin for Vite and configuring tokens.
